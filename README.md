# Scanner Front (Next.js)

Фронтенд на `Next.js` для OCR-сценария:
- загрузка 3 документов (паспорт, прописка, выписка ЕГРН),
- просмотр результата распознавания,
- ручная корректировка полей,
- генерация и скачивание договора `.docx`.

## Возможности

- Единый запуск сканирования через endpoint бэкенда.
- Редактирование данных паспорта и ЕГРН перед генерацией.
- Ручные поля для договора:
  - адрес заказчика,
  - основание собственности,
  - email заказчика,
  - телефон заказчика.
- Кнопка генерации договора и мгновенное скачивание файла.

## Запуск

```bash
npm install
npm run dev
```

Откройте: `http://localhost:3000`

## Настройка API

Используется переменная окружения:

- `NEXT_PUBLIC_API_BASE_URL` — адрес бэкенда FastAPI  
  (по умолчанию `http://127.0.0.1:8000`).
